# Project Overview

## Purpose
DEVRC is a personal development environment configuration repository designed for Ubuntu 20 LTS and NixOS systems. It provides a complete, reproducible development environment setup using Nix and home-manager.

## Key Features
- Declarative environment configuration using Nix
- Home-manager integration for user environment management
- Multi-language support (Go, Python, Rust, Node.js, Lua, C#, Perl, Solidity, Cue, Nix, YAML, and more)
- Neovim configuration with extensive plugin ecosystem
- Shell configuration (zsh, bash) with customizations
- Git, tmux, fzf, direnv, alacritty, k9s, ranger integrations
- Docker installation and setup

## Target Users
Innovation Upstream developers needing consistent development environments across machines

## System Requirements
- Ubuntu 20 LTS or NixOS
- May work on other Linux distributions with modifications
